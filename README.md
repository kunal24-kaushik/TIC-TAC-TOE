# 🎮 Tic Tac Toe Studio

A modern **Tic Tac Toe** game built using **Python**, **Tkinter**, and **Pillow (PIL)**. This project offers an attractive GUI, multiple AI difficulty levels, score tracking, smooth animations, and a premium arcade-style experience.

## 📌 Features

- 🎨 Modern and interactive graphical interface
- 👤 Player name customization
- 🤖 Three AI difficulty levels:
  - Easy (Random Moves)
  - Hard (Smart Moves)
  - Impossible (Minimax Algorithm)
- 🏆 Live score tracking
  - Wins
  - Losses
  - Draws
- 🎯 Animated button hover effects
- 🌈 Random colors for X and O every game
- 📊 Premium score cards
- 🔄 Play Again and New Player options
- 🖼️ Background image with blur effect
- 💬 Custom popups instead of default message boxes

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/tic-tac-toe-studio.git
```

### 2. Move into the project folder

```bash
cd tic-tac-toe-studio
```

### 3. Install dependencies

```bash
pip install pillow
```

### 4. Run the project

```bash
python main.py
```

---

## 🎮 Gameplay

1. Enter your player name.
2. Select the difficulty level.
3. Click **Start Game**.
4. Play as **X**.
5. Computer plays as **O**.
6. Score updates automatically after each match.

---

## 🧠 AI Levels

### Easy
- Computer selects random available positions.

### Hard
- Tries to win first.
- Blocks the player's winning moves.

### Impossible
- Uses the **Minimax Algorithm**.
- Plays perfectly and cannot be beaten.
